## 📌 Descripción de la aplicación

Esta aplicación móvil fue desarrollada en Android Studio utilizando Java y SQLite.

Permite gestionar una lista de tareas mediante las siguientes funcionalidades:

- Registrar tareas
- Buscar tareas por ID
- Editar tareas
- Eliminar tareas
- Visualizar todas las tareas en una lista (RecyclerView)

Cada tarea contiene:
- ID
- Título
- Descripción
- Estado (0 = pendiente, 1 = completada)

## 📂 Estructura básica

- MainActivity.java → lógica principal
- Tarea.java → modelo de datos
- TareaAdapter.java → adaptador del RecyclerView
- AdminSQLiteOpenHelper.java → base de datos
- activity_main.xml → interfaz principal
